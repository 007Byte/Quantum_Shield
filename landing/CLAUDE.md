# Landing Page

## Design Skill
All UI/UX work MUST follow the master design skill at `../.claude/skills/SKILL.md`. Reference it before building or modifying any component.

## Stack
- Next.js 16 (App Router, React 19)
- Tailwind CSS v4 (uses `@theme inline` in CSS, NOT tailwind.config.ts)
- Framer Motion 12+
- TypeScript strict

## Brand Direction
Dark luxury cyberpunk. Space Grotesk display font. Deep purple (#753cff) accent, cyan (#22D3EE) secondary. Seamless section transitions — no hard borders between sections.

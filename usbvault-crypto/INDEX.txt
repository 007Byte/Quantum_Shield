CRYPTOGRAPHIC SPECIFICATIONS
================================================================================

Key Derivation (Argon2id)
  Memory: 65,536 KB (64 MB)
  Time cost: 3 iterations
  Parallelism: 4 lanes
  Output: 64 bytes (32B encryption + 32B HMAC)

Symmetric Encryption - XChaCha20-Poly1305
  Key size: 256 bits (32 bytes)
  Nonce size: 24 bytes (random per operation)
  Tag size: 128 bits (16 bytes)
  Security level: 256-bit

Symmetric Encryption - AES-256-GCM-SIV
  Key size: 256 bits (32 bytes)
  Nonce size: 12 bytes
  Tag size: 128 bits (16 bytes)
  Security level: 256-bit

Streaming AEAD
  Chunk size: 64 KB (65,536 bytes)
  Chunk nonce: base_nonce XOR chunk_index (8B)
  Metadata chunk: Encrypted separately

Asymmetric Sharing (X25519 Sealed Box)
  Algorithm: X25519 ECDH
  Key size: 256 bits (32 bytes)
  Nonce: 24 bytes (XChaCha20-Poly1305)
  Format: ephemeral_pk (32) || nonce (24) || ciphertext || tag (16)

Authentication (SRP-6a)
  Protocol: Zero-knowledge proof
  Password transmission: None (no plaintext sent)
  Hash: SHA-256

SECURITY FEATURES
================================================================================

Confidentiality
  • 256-bit symmetric encryption keys
  • Authenticated encryption (AEAD) with 128-bit tags
  • Perfect forward secrecy via ephemeral X25519

Integrity
  • 128-bit authentication tags per AEAD operation
  • HMAC-SHA256 for vault header validation
  • Per-chunk verification in streaming mode

Authenticity
  • SRP-6a zero-knowledge password authentication
  • No plaintext password transmission
  • HMAC-based password verification

Memory Security
  • Zeroize wrappers for all sensitive data
  • Automatic secure cleanup on drop
  • Platform-specific mlock/munlock support (Linux)

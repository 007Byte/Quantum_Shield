# Archive

Historical artifacts from USBVault Enterprise development phases 1-5.
These files are kept for reference but are **not part of the active codebase**.

## Contents

### `reports/`
Generated dashboards, gap analyses, QAV audit reports, roadmap trackers,
and security analysis snapshots produced during development.

### `implementation-notes/`
Working documents from implementation phases: fix checklists, consolidation
manifests, debug prompts, and E2E verification reports.

### `assets/`
Concept art, logo drafts, and UI aspiration images not used by the app.
Production assets live in `usbvault-app/assets/`.
